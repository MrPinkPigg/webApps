INTRUCTIONS TO GET MY THING WORKING

1. unzip the file into htdocs under your apache install
2. First, start your server with MySQL.
3. Navigate a browser to localhost/examples/install.php
4. You should see a bunch of messages about database and table creation
5. Done! Navigate your browser to localhost/examples/index.html and login using test/test
